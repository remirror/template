## Installation

```bash
# yarn
yarn add template-package

# pnpm
pnpm add template-package

# npm
npm install template-package
```

<br />

## Usage

The following code creates an instance of this extension.

```ts
import { printTemplate } from 'template-package';

printTemplate(); // => 'tEmPlATe'
```
